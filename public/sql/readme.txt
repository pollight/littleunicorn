Архив содержит информацию о городах, регионах и странах встречающихся 
в базе данных Sypex Geo City Max.

Данные представлены в формате TSV (Tab Separated Values — значения, 
разделённые табуляцией) в кодировке UTF-8. Для импорта в MySQL можно 
воспользоваться sxgeo.sql файлом. С его помощью создается 3 таблицы 
с данными.

Перед выполнением запроса нужно указать путь к файлам city.tsv, 
region.tsv, country.tsv в запросах LOAD DATA INFILE.
